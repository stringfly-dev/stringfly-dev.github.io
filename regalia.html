Regalia

PROJECT OVERVIEW

Regalia is an AI Agent platform that lets users build functional AI agents through structured workflows, reusable knowledge, and controllable actions. I designed the end-to-end experience—from branding to agent creation and discovery to make the platform clearer, more reliable, and easy to use.

THE CHALLENGE

The prompt-based framework of the existing Agent Launchpad fails to provide users with control over the agent's actions, leading to errors and the inability to create functional agents that perform tasks reliably

DESIGN APPROACH

Branding & Design System

I designed the logo, branding, and complete design system for Regalia, including color palettes, typography and UI components, to maintain visual consistency across all platforms.

Graphic Templates for Marketing

I created graphic templates for social media, promotional content to ensure consistent branding and faster asset production for the marketing team.

Discover AI Agent

I designed the Discover page to help users explore AI agents efficiently through categories, trending filters, and detailed performance metrics. This improves discoverability and helps users quickly identify agents that match their needs.

Agent Creation

I designed the end-to-end Agent Creation flow, guiding users from setting up the agent’s identity to adding knowledge sources and building structured workflows. The flow includes workflow templates that help beginners get started quickly. Users can preview each step inputs, actions, and outputs to ensure accuracy before launching their agent.


Agent Profile

I designed the agent profile to highlight metrics such as usage, capabilities, workflows, and performance history. This helps users understand an agent’s strengths and determine whether it fits their use case.

User Profile Modal

I also designed the user profile to showcase created agents, social account, and account settings.

KEY LEARNING & REFLECTIONS

  • I gained deeper understanding of AI agent design, especially how workflows, actions, and knowledge inputs affect reliability.
  • Building an end-to-end design system and CI branding
  • Close collaboration with engineers improved my understanding of the tech side and agent execution pipelines.
